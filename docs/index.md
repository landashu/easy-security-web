## ✨ 当前最新版本

```
// 我们建议使用最新版本
<dependency>
  <groupId>com.aizuda</groupId>
  <artifactId>easy-security-boot-starter</artifactId>
  <version>2.0.4</version>
</dependency>
```
::: tip
当你下载依赖失败，可能是由于依赖还没有审核通过，你可以在 maven 仓库找低于当前版本的依赖使用
:::
